(static) friendUser(options) → {Promise}

Source:
  • tbapi.js, line 372

Creates a relationship between a user and a subreddit. This is used for:

  • Banning users
  • Wikibanning users
  • Muting users
  • Adding moderators
  • Adding wiki contributors
  • Accepting moderator invitations
Parameters:
Name Type Description
options object
Properties
Name Type Attributes Description
user string

The user to apply the relationship to
action string

The string for the desired action (see https://www.reddit.com/dev/api#POST_api_friend for a list)
subreddit string

The sub to apply the relationship in
banReason string <optional>

If banning, the private mod note
banMessage string <optional>

If banning, the note sent to the user
banDuration number <optional>

If banning, the length of the ban (0 or undefined for a permanent ban)
banContext string <optional>

If banning, a fullname pointing to the link or comment the user is being banned for
Returns:

Resolves to the JSON response body or rejects with a jqXHR object
